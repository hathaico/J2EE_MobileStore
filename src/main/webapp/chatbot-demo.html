❓ Các Câu Hỏi Thường Gặp (FAQs)

Q1: Bot tư vấn được những gì?

A: Bot có thể giúp bạn:

  • Tìm điện thoại phù hợp với ngân sách
  • So sánh thông số kỹ thuật giữa các sản phẩm
  • Kiểm tra giá, khuyến mãi và tồn kho
  • Tra cứu phí vận chuyển theo khu vực
  • Trả lời các câu hỏi kỹ thuật cơ bản
  • Giới thiệu các chương trình ưu đãi

Q2: Bot có thể lập đơn hàng không?

A: Tại thời điểm này, bot chỉ tư vấn và cung cấp thông tin. Để mua hàng, bạn cần:

  • Truy cập website để lập đơn trực tiếp
  • Gọi hotline để nhân viên hỗ trợ
  • Chat với nhân viên CSKH qua bot

Q3: Data sản phẩm có được cập nhật thực tế?

A: Hiện tại đang sử dụng dữ liệu mẫu. Để tích hợp với hệ thống thực, bạn cần:

  • Kết nối với database sản phẩm thực tế (MySQL, PostgreSQL)
  • Đồng bộ hóa giá và kho hang realtime
  • Tích hợp API thanh toán và giao hàng

Q4: Làm sao để tích hợp bot vào website?

A: Xem phần "Hướng Dẫn Tích Hợp" bên dưới.

Q5: Bot có hỗ trợ các ngôn ngữ khác không?

A: Hiện tại chỉ hỗ trợ Tiếng Việt. Có thể mở rộng hỗ trợ các ngôn ngữ khác bằng cách:

  • Dịch tất cả tin nhắn vào i18n
  • Sử dụng Google Translate API hoặc service tương tự
  • Tích hợp NLP multi-language

Q6: API endpoint của bot ở đâu?

A: API được expose tại:

POST /api/chatbot/message
GET /api/chatbot/health
GET /api/chatbot/greeting

🔧 Hướng Dẫn Tích Hợp Chatbot Vào Website

Bước 1: Include CSS & JS trong HTML
<head>
  <link rel="stylesheet" href="assets/css/chatbot.css">
</head>
<body>
  <!-- Page content -->
  <script src="assets/js/chatbot.js"></script>
</body>
Bước 2: Thêm HTML Widget
<body>
  <div id="chatbot-widget" class="chatbot-container">
    <!-- Widget HTML từ chatbot.html -->
  </div>
</body>
Bước 3: Cấu Hình API Base URL
// Trong chatbot.js, cập nhật CHATBOT_API
const CHATBOT_API = 'http://your-domain:8080/api/chatbot';
// hoặc nếu cùng domain
const CHATBOT_API = '/api/chatbot';
Bước 4: Bật CORS trong Spring Boot
// Sẽ tự động enable trong ChatBotController
@CrossOrigin(origins = "*")
Bước 5: Test API
curl -X POST http://localhost:8080/api/chatbot/message \
  -H "Content-Type: application/json" \
  -d '{"userMessage":"Xin chào"}'
